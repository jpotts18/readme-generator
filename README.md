# Readme Generator

The best way to generate the documentation your boss always wanted and you hesitated to write.

## System Dependencies

* Ruby 2.0+

## Getting Started

1. Clone the Repo ```git clone [this repo]```
1. Open up editor ```ruby readme.rb```
1. Follow the prompt and enjoy!
